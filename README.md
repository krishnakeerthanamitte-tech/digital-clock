# digital-clock
Digital clock (12-Hour Format)
A simple and visually pleasing 12-Hour digital clock built using HTML,CSS, and JavaScript, featuring a nature-themed background image for a calm and aestheti look.

**Features** 
  - Displays time in 12-hour format (AM / PM)
  - Updates every second in real time
  - Nature background image for a relaxing UI
  - Clean and beginner-friendly code

**Technologies Used**
  - HTML for structure of the clock
  - CSS for styling and background image
  - JavaScript for time logic and real-time updates

**How It Works**
  - Uses JavaScript's Date() to get the current time
  - Converts time into 12-hour format
  - Determines AM / PM
  - SetInterval() updates the time every second
  - DOM manipulation displays the updated time on the screen
